# 3.5 Character App

## Overview
This project is an open-source character sheet manager designed for use with **3.5-edition compatible tabletop fantasy role-playing games**.

The application allows users to create, manage and update character sheets.
This project is intended as a **rules agnostic character management tool**, with optional support for **Open Game Content** where legally permitted.
